# Copiloto-IA-RH
Este projeto foi desenvolvido para a Faculdade de Tecnologia Rocketseat, na disciplina Fundamentos de IA com foco em Inteligência Artificial Generativa. A solução consiste na automação da criação de textos institucionais, como avisos e e-mails corporativos, promovendo padronização, agilidade e eficiência na comunicação
